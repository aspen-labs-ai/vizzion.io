## Implementation Details

### Design Specifications
- **Height:** 48px (40px on mobile)
- **Format:** SVG (scalable, lightweight)
- **Style:** Brand colors with grayscale filter
- **Hover Effect:** 
  - Removes grayscale filter
  - Increases opacity to 100%
  - Subtle lift animation (translateY -4px)
  - Scale increase (1.05x)
